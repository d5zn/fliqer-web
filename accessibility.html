Fliqer is accessible to all users

Our Commitment to Accessibility

We strive to make Fliqer as accessible as possible for all users, regardless of their abilities or preferences.

Technical Features

VoiceOver and Dynamic Text

  • Semantic Elements: All interface elements have proper roles and labels for VoiceOver
  • Navigation: Logical sequence of elements for convenient navigation
  • Dynamic Text: Automatic adaptation of font size to system settings
  • Alternative Text: Descriptions for all graphical elements

Color Scheme

Color Contrast and Accessibility
  • Text to background contrast ratio: minimum 4.5:1
  • Color information is duplicated with text and icons
  • Support for iOS color filters
  • Adaptation to "Invert Colors" settings

Gesture Control

  • Alternative Gestures: All functions are accessible through different interaction methods
  • Larger Touch Areas: Buttons and controls have sufficient size
  • Customizable Gestures: Support for user-defined iOS gesture settings

iOS Compatibility

Supported Accessibility Technologies

  • VoiceOver: Full screen reader support
  • Zoom: Compatibility with screen magnification feature
  • Switch Control: Support for switch-based control
  • AssistiveTouch: Works with virtual buttons
  • Guided Access: Restricts access to specific areas
  • Speak Screen: Reads screen content

Settings and Personalization

Adaptive Themes

Automatic Adaptation
Fliqer automatically switches between light and dark themes depending on iOS system settings, ensuring optimal contrast and eye comfort.

Interface Settings

  • Font Size: Automatic text scaling
  • Font Weight: Support for user-defined settings
  • Animations: Compliance with "Reduce Motion" setting
  • Contrast: Adaptation to system settings
